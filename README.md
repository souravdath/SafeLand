# SafeLand
## AI-Based Flood Risk Prediction System for Kerala

SafeLand is a machine-learning-based system that predicts flood risk levels
for house construction sites in Kerala using historical flood data and
geographical features.

## Features
- Location-based flood risk assessment
- Flood risk classification (Low / Medium / High)
- Supports safe house construction decisions

## Tech Stack
- Python
- Scikit-learn
- Flask
- HTML, CSS, JavaScript
